# 💳 Simulador de Conta Bancária

Projeto desenvolvido em Java para simular operações básicas de uma conta bancária através do terminal.

O sistema permite consultar saldo, realizar transferências e receber valores, utilizando estruturas fundamentais da linguagem Java.

---

## 🚀 Tecnologias utilizadas

- Java
- IntelliJ IDEA

---

## 📋 Funcionalidades

- Consultar saldo
- Transferir valores
- Receber depósitos
- Validação de saldo insuficiente
- Menu interativo no terminal

---

## 🧠 Conceitos praticados

- Variáveis
- Estruturas condicionais (if/else)
- Estruturas de repetição (while)
- Entrada de dados com Scanner
- Operadores aritméticos
- Lógica de programação

---

## 🎯 Objetivo do projeto

Praticar conceitos fundamentais de Java através da implementação de um sistema simples de gerenciamento bancário.
